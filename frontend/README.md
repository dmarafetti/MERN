# Form Constraint Component

### Installation

```sh 
$ npm install
```

###  Development
Automatically run webpack on any file change.


```sh 
$ npm run watch 
```

Same as 'watch' but with http server.

```sh 
$ npm run server 
```



###  Distribute

```sh 
$ npm run build
```


###  Publish

```sh 
$ npm publish 
```
