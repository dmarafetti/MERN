# Form Constraint Component

### Installation

```sh 
$ npm install
```

###  Development
Automatically run webpack on any file change.


```sh 
$ npm run watch 
```

Same as 'watch' but with http server.

```sh 
$ npm run server 
```

Run frontend app separately. A few env variables must be declared

```sh 
$ export UI_BUILD_MODE=development
```





###  Distribute

```sh 
$ npm run build
```


###  Publish

```sh 
$ npm publish 
```
